----------------
| PREREQUISITE |
----------------

-> Download SAP COMMERCE 2211 JDK21 Suite from Software Center.
    link -: https://me.sap.com/softwarecenterviewer/73555000100800001224/INST
-> Download Spartacus Sample Data
    link -: https://github.com/SAP-samples/cloud-commerce-sample-setup
-> Install JDK21 
    link -: https://sapmachine.io/
-> Install 7Zip
-> Install Git & Git bash
-> Install NodeJS -: 22.22.0, Angular Cli 21.1.0, NPM - 11.10.1
    npm install -g @angular/cli@21.1.0
    ng version
-> Install IDE/Editor - Intellij, VS Code, Notepad++
------------------------------------------------------------------------

-------------------------
| Initial SetUp Process |
-------------------------

-> Extract SAP COMMERCE SUITE in your desire folder with the help of 7zip 
    (Note -: Url folder name/path should not contain spaces)

-> In hybris/bin: Create the folder custom and Extract, Move the entire spartacussampledata folder to hybris/bin/custom.

-> Move the custom.properties file from spartacussampledata/resources/installer/customconfig to 
    Hybris2211JDK21SPA/installer/customconfig.

-> In the Hybris2211JDK21SPA/installer/customconfig/custom.properties file, add the following line:
    authserver.oauthclientdetails.loginpageuri.allowed.hosts=localhost
    cookies.SameSite.enabled=true
    acceleratorservices.payment.sopmock.enabled=true
    initialpassword.admin=nimda
    smartedittools.only.build.once=true
    commercewebservices.api.restrictions.disabled.endpoints=

-> In the installer/recipes folder: Duplicate the cx_features_enabled folder & Change the name of 
    the duplicate folder to cx_features_enabled_for_spa.

-> In the new cx_features_enabled_for_spa folder:
    Open build.gradle with a text editor.
    Uncomment // extName 'spartacussampledata' in the list of extensions.
    
    Comment out the following test extensions:
    // extName 'b2bocctests'
    // extName 'b2bpunchoutocctests'
    // extname 'chineseaddressocctests'
    // extName 'commercewebservicestests'
    // extName 'configurablebundleocctests'
    // extName 'customerticketingocctests'
    // extName 'sapproductconfigocctests'
    // extName 'textfieldconfiguratortemplateocctests'
    // extName 'yocctests'

| Installion Process |
----------------------

-> Open a terminal or command prompt window inside the installer folder.
-> Set up the recipe using the following command:
    Windows -: install.bat -r cx_features_enabled_for_spa

    Windows -: install.bat -r cx_features_enabled_for_spa initialize

-----------------------
| SPARTACUS SPA SETUP |
-----------------------

NodeJS - 22.22.0
NVM - 11.10.1
Angular Cli - 21.1.0


*****SPA Initial SetUp Process*****

-> Create folder 'spartacus' inside Hybris2211JDK21SPA/ folder & 
    Open a terminal or command prompt window at that location Hybris2211JDK21SPA/spartacus.

-> Create your project by running following command (Hit Enter for selecting None if asked for AI TOOLS Configure) -:
    ng new spartacus-app --style=scss --ssr=false --zoneless=false --file-name-style-guide=2016

-> Access project -:
    cd spartacus-app

*****Install Spartacus Libraries from the Repository Based Shipment Channel (RBSC)*****

-> Create .npmrc file inside project -: Hybris2211JDK21SPA/spartacus/spartacus-app
-> Open .npmrc file with a text editor and add RBSC auth key, Example -:

@spartacus:registry=https://73554900100900004337.npmsrv.base.repositories.cloud.sap/
//73554900100900004337.npmsrv.base.repositories.cloud.sap/:_auth=<YourAuthKey From RBSC>
always-auth=true

-> Copy the .npmrc file and past to your user account -: C:\Users\<<YourAccount>>

-> Run the following command to install spartacus storefront using schematics:
    ng add @spartacus/schematics@~221121.7

    When Promted -: Would you like to share pseudonymous usage data -> Enter N (No)
    When Promted -: for Installing @spartacus/schematics -> Enter Y (Yes)
    When Promted -: Which spartacus set would you like to set up -> Hit Enter (Keep Default)

-> Install Dependencies
    npm install

*****Spartacus Server issue fix*****

-> Added "src" inside stylePreprocessorOptions.includePaths in angular.json.
    "stylePreprocessorOptions": {
        "includePaths": ["src", "node_modules"]
    },

-> Replaced all relative imports like (present in ..spartacus-app\src\styles\spartacus):
    @import "..\..\styles-config";
    with:

    @import "styles-config";

-> Start Server
    ng serve --ssl true
    
    SPARTACUS Site Url -: https://localhost:4200/
