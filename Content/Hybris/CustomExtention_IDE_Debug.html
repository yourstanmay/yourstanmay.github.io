        CUSTOM EXTENSION | DEBUG CONFIGURATION | CODE SETUP IN INTELLIJ

        1. Delete 'yb2bacceleratorstorefront' folder under custom folder.
        2. Open localextentions.xml file and comment out -: <!-- <extension name='yb2bacceleratorstorefront' /> -->
        3. Navigate to platform folder and open cmd. Execute below command -: 
               - > setantenv.bat
               - > ant modulegen
               - > template? No need provide input , just press enter
               - > name? yts
               - > package?com.yts
        
         **Above command generates 7 extensions here {HYBRIS_BIN_DIR}\custom\yts
                    ytscore
                    ytsbackoffice
                    ytstest
                    ytsfulfilmentprocess
                    ytsfacades
                    ytsstorefront
                    ytsinitialdata
        
        4.  Open localextentions.xml and add bellow extentions
        
        <!-- yts Specific Extensions -->
            <extension name='ytsfulfilmentprocess' />
            <extension name='ytscore'/>
            <extension name='ytsinitialdata'/>
            <extension name='ytsfacades'/>
            <extension name='ytsbackoffice'/>
            <extension name='ytsstorefront'/>
                 
                **Comment out bellow extensions 
                <!-- <extension name='yacceleratorbackoffice' />
                    <extension name='yacceleratorcockpits' />
                    <extension name='yacceleratorcore' />
                    <extension name='yacceleratorfacades' />
                    <extension name='yacceleratorinitialdata' /> 
                    <extension name='yacceleratorstorefront' /> -->
                    <!-- <extension name='yacceleratorordermanagement' /> -->
        
        
        5. Open local.properties and add below property to configure debug 
            #Enabling Debug Option
              tomcat.debugjavaoptions=-Xdebug -Xnoagent -Xrunjdwp:transport=dt_socket,server=y,address=8002,suspend=n
        
           **Replace the below properties
              website.electronics.http=http\://electronics.local\:9001/ytsstorefront
              website.electronics.https=https\://electronics.local\:9002/ytsstorefront
              website.apparel-de.http=http\://apparel-de.local\:9001/ytsstorefront
              website.apparel-de.https=https\://apparel-de.local\:9002/ytsstorefront
              website.apparel-uk.http=http\://apparel-uk.local\:9001/ytsstorefront
              website.apparel-uk.https=https\://apparel-uk.local\:9002/ytsstorefront
              website.powertools.http=http\://powertools.local\:9001/ytsstorefront
              website.powertools.https=https\://powertools.local\:9002/ytsstorefront
        
        6. Trigger "ant clean all"
        
        7. hybrisserber.bat debug
        
        8. Navigate to HAC and perform -: Update Running System
        
        9. Add bellow mentioned URL Patterns
            (?i)^https?://api\.hybrisdev\.com(:[\d]+)?/rest/.*$
            (?i)^https?://localhost(:[\d]+)?/rest/.*$
            ^.*$
            (?i)^https?://[^/]+(|/.*|\?.*)$
            (?i)^https?://[^/]+(/[^?]*)?/powertools/\w+\??.*$
        
        10. Hit below URL to access the site
             https://apparel-uk.local:9002/ytsstorefront/en/
             https://localhost:4200
