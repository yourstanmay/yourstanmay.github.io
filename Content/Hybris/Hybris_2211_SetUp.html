===========================
HYBRIS 2211 SETUP
===========================

1. Install JDK 17 -: https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html
   Download SAP Hybris Suite 2211 from URL -: 
   https://me.sap.com/softwarecenter/search/SAP%2520Commerce%2520Cloud%2520V2
    
2. Create a folder SAPHybris2211 and extract the ZIP under this folder.
    
3. Open CMD inside Installer folder and run the command : install.bat -r cx setup -A initAdminPassword=nimda
	
4. Initialize the system using the following command: install.bat -r cx initialize setup -A initAdminPassword=nimda
	
5. Open CMD inside hybris\bin\platform and start your server using command : hybrisserver.bat
